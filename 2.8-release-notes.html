SchoolTool 2.8 Release Notes¶

SchoolTool 2.8 is a feature release for new deployments or those ready to upgrade their 2.6 installation.

See SchoolTool 2.8 And Beyond for plans regarding this and future releases.

New features¶

  • Relationships (enrollment, group membership) have explicit states (e.g., pending, enrolled, withdrawn) with explicit start dates

  • Enrollment modes

  • Basic parent access

  • Levels for courses and persons

  • Transcript reports

See Changes in SchoolTool 2.8 for introduction to the new relationships.

Availability¶

It is available for all supported Ubuntu releases in a new 2.8 ppa

When to Upgrade¶

We recommend new users of SchoolTool begin with the 2.8 release.

Users of SchoolTool 2.8.0 should upgrade to 2.8.3 at a convenient time.

Users of SchoolTool 2.6 should upgrade to 2.8.3 after testing against their database and reviewing the changes in functionality.
